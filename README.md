Este repositorio contiene la **Evaluación Final del Módulo 2** del bootcamp de Adalab, realizada por **Patricia Anaya Morales**.

El proyecto consiste en la extracción de datos de películas desde una API, la creación de una base de datos MySQL y la realización de consultas analíticas.

### Cómo Arrancar el Proyecto

El código completo se encuentra en dos *notebooks* de Jupyter:
- **`bda-modulo-2-evaluacion-final-PatriciaAnaya.ipynb`**: Extracción de datos de la API e inserción en la base de datos.
- **`bda-modulo-2-evaluacion-final-PatriciaAnaya-consultas.ipynb`**: Consultas analíticas sobre los datos.

Para ejecutar:

1. Abre el *notebook* **`bda-modulo-2-evaluacion-final-PatriciaAnaya.ipynb`** en Jupyter y ejecuta todas las celdas para crear la base de datos e insertar los datos.
2. Abre el *notebook* **`bda-modulo-2-evaluacion-final-PatriciaAnaya-consultas.ipynb`** para ejecutar las consultas analíticas.
