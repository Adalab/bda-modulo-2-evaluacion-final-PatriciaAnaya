Este repositorio contiene la **Evaluación Final del Módulo 2** del bootcamp de Adalab, realizada por **Patricia Anaya Morales**.

El proyecto consiste en la extracción de datos de películas desde una API, la creación de una base de datos MySQL y la realización de consultas analíticas.

### Contenido del Repositorio

* **`bda-modulo-2-evaluacion-final-PatriciaAnaya.ipynb`**: Notebook de Jupyter para la **extracción de datos de la API e inserción en la base de datos** (la base de datos principal del proyecto).
* **`bda-modulo-2-evaluacion-final-PatriciaAnaya-consultas.ipynb`**: Notebook de Jupyter con las **consultas analíticas** sobre los datos del proyecto.
* **`Ejercicio 2.sql`**: Archivo con una colección de **consultas de SQL** adicionales, realizadas sobre el esquema de la base de datos **Sakila**.

### Cómo Arrancar el Proyecto

#### Para la Base de Datos y Consultas Analíticas del Proyecto:

1.  Abre el *notebook* **`bda-modulo-2-evaluacion-final-PatriciaAnaya.ipynb`** en Jupyter y ejecuta todas las celdas para crear la base de datos e insertar los datos del proyecto.
2.  Abre el *notebook* **`bda-modulo-2-evaluacion-final-PatriciaAnaya-consultas.ipynb`** para ejecutar las consultas analíticas.

#### BONUS:

* El archivo **`Ejercicio 2.sql`** contiene código SQL que requiere que tengas la base de datos **Sakila** instalada y configurada. Se puede ejecutar directamente en un entorno MySQL conectado a esta base de datos.
